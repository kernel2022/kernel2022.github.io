Tableau Day2: 可视化入门图形制作

2022-01-05 Tableau 技能 14 天打卡训练营 ， tableau

▶ Day2 课程内容
在 Tableau 中绘制折线图、柱形图、甘特图、饼图、圆环图与树状图的思路与实践方法。

▶ Day2 练习数据（复制链接至浏览器获取）

目录

折线图

  1. 到文件
    • Microsoft Excel
  2. 示例-超市.xls
  3. 工作表
  4. 拖拽 订单，查看数据源
  5. 双击左下角 工作表1
    • 重命名为 折线图
  6. 拖拽 销售额 到 行
  7. 拖拽 订单日期 到 列
  8. 工具栏：标准 -> 整个视图

柱形图

  1. 点击左下角 新建工作表
    • 重命名为 柱形图
  2. 拖拽 销售额 到 行
  3. 拖拽 子类别 到 列

甘特图

  1. 点击左下角 新建工作表
    • 重命名为 甘特图
  2. 左侧下拉：创建计算字段
    • 重命名为 间隔天数
  3. 下拉选择：日期
  4. 双击：DATEDIFF
    • 第一个参数 日期部分：'day'
    • 第二个参数 开始日期：拖拽 订单日期 到 开始日期
    • 第三个参数 结束日期：拖拽 发货日期 到 结束日期
    • DATEDIFF('day', [订单日期], [发货日期])
    • 点击 应用
    • 点击 确定
  5. 拖拽 订单ID 到 行
  6. 拖拽 订单ID 到 筛选器
  7. 选择：从列表中选择
  8. 点击：无
  9. 选中前 10 个
  10. 点击 应用
  11. 点击 确定
  12. 拖拽 订单日期 到 列
  13. 右击：订单日期
    • 选中：精确日期
  14. 拖拽 间隔天数 到 大小
  15. 右击 间隔天数
    • 选中：度量（平均值）
  16. 拖拽 发货日期 到 详细信息
  17. 右击：发货日期
    • 选中：精确日期

饼图

  1. 点击左下角 新建工作表
    • 重命名为 饼图
  2. 下拉：标记
    • 选中：饼图
  3. 拖拽 类别 到 颜色
  4. 工具栏：标准 -> 整个视图
  5. 拖拽 数量 到 角度
  6. 拖拽 类别 到 标签
  7. 拖拽 数量 到 标签
  8. 右击：总和（数量）
    • 选中：快速表计算
      • 选中：合并百分比

圆环图

  1. 点击左下角 新建工作表
    • 重命名为 圆环图
  2. 下拉：标记
    • 选中：饼图
  3. 选中 行
    • 输入 0 回车
    • 再输入 0 回车
  4. 工具栏：标准 -> 整个视图
  5. 选中上方：总和0
    • 拖拽 类别 到 颜色
    • 拖拽 数量 到 角度
    • 选中：大小
      • 放大一些
  6. 选中下方：总和0
    • 右击：Y轴
      • 选中：双轴
    • 选中：颜色
      • 选中：白色
    • 选中：大小
      • 放大一些
  7. 选中上方：总和0
    • 拖拽 类别 到 标签
    • 拖拽 数量 到 标签
    • 右击：总和（数量）
      • 选中：快速表计算
        • 选中：合并百分比

树形图

  1. 点击左下角 新建工作表
    • 重命名为 树形图
  2. 下拉：标记 -> 方形
  3. 拖拽 客户名称 到 详细信息
  4. 拖拽 销售额 到 大小

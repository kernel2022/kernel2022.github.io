▶ Day3 课程内容
学习如何使用参数，实现业务问题预警与自定义选择指标。

▶ Day3 练习数据（复制链接至浏览器获取）

达标判断

  1. 到文件
    • Microsoft Excel
  2. 示例-超市.xls
  3. 工作表
  4. 拖拽 订单，查看数据源
  5. 双击左下角 工作表1
    • 重命名为 达标判断
  6. 拖拽 订单日期 到 列
    1. 下钻：年-月
  7. 拖拽 销售额 到 行
  8. 工具栏：标准 -> 整个视图
  9. 拖拽 订单日期 到 列
  10. 下拉：标记
  • 选择：条形图
  1. 点击左上角 分析（在数据旁边）
    1. 拖拽 常量线 到 右侧 表
    2. 输入 400000 （四十万）
  2. 点击左上角 数据
  3. 搜索 旁边的 下拉
  4. 选择 创建计算字段
  5. 重命名：达标判断
IF SUM([销售额]) >= 400000 THEN '达标'
ELSE '不达标'
END
  1. 拖拽 达标判断 到 颜色
  2. 右侧下拉：编辑颜色
    1. 不达标：红
    2. 达标：绿

变动目标

  1. 右击 达标判断 工作表
    1. 点击 复制 （复制为交叉表上方）
    2. 重命名为 变动目标
  2. 右击 常量线
    1. 点击 移除
  3. 点击 分析
    1. 拖拽 平均线 到 右侧 表
    2. 右击：编辑
    3. 线 / 值 / 下拉：创建新参数
      1. 名称：变动目标
      2. 值范围
        1. 最小值：300,000（三十万）
        2. 最大值：500,000（五十万）
        3. 表长： 5000（五千）
    4. 标签：值
  4. 右击：左下角 参数 / 变动目标
    1. 选择：显示参数
  5. 右击：表 / 达标判断
    1. 选择：编辑
IF SUM([销售额]) >= [变动目标] THEN '达标'
ELSE '不达标'
END

动态度量

  1. 左下角新建 工作表
    • 重命名为 动态度量
  2. 拖拽 子类别 到 列
  3. 拖拽 销售额 到 行
  4. 工具栏：标准 -> 整个视图
  5. 搜索 旁边的 下拉
  6. 选择 创建参数
    1. 名称：度量选择
    2. 字符类型：字符串
    3. 允许的值：列表
    4. 值列表：值
      1. 销售额
      2. 利润
      3. 数量
  7. 右击：左下角 参数 / 度量选择
    1. 选择：显示参数
  8. 搜索 旁边的 下拉
  9. 选择 创建计算字段
    1. 重命名：动态度量
IF [度量选择] == '销售额' THEN [销售额]
ELSEIF [度量选择] == '利润' THEN [利润]
ELSEIF [度量选择] == '数量' THEN [数量]
END
  1. 拖拽 动态度量 到 行 替换 总和（销售额）

动态维度

  1. 左下角新建 工作表
    • 重命名为 ** 动态维度**
  2. 拖拽 子类别 到 列
  3. 拖拽 销售额 到 行
  4. 工具栏：标准 -> 整个视图
  5. 搜索 旁边的 下拉
  6. 选择 创建参数
    1. 名称：维度选择
    2. 字符类型：字符串
    3. 允许的值：列表
    4. 值列表：值
      1. 子类别
      2. 区域
      3. 细分
  7. 右击：左下角 参数 / 维度选择
    1. 选择：显示参数
  8. 选择 创建计算字段
    1. 重命名：动态维度
IF [维度选择] == '子类别' THEN [子类别]
ELSEIF [维度选择] == '区域' THEN [区域]
ELSEIF [维度选择] == '细分' THEN [细分]
END
  1. 拖拽 动态度量 到 列 替换 子类别

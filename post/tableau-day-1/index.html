绘制柱形图

  1. 点击左下角 新建工作表
  2. 重命名为 柱形图
  3. 双击 销售额
  4. 双击 子类别
  5. 拖拽 销售额 到 标签
  6. 拖拽 利润 到 颜色

先双击维度，生成表格
先双击度量，生成图

绘制拆线图

  1. 点击左下角 新建工作表
  2. 重命名为 拆线图
  3. 双击 销售额
  4. 双击 订单日期
  5. 工具栏：标准 -> 整个视图
  6. 下钻：列 -> 年（订单日期） -> 季度（订单日期）
  7. 下钻：列 -> 季度（订单日期） -> 月（订单日期）
  8. 移除：季度（订单日期）

绘制地图

  1. 点击左下角 新建工作表
  2. 重命名为 地图
  3. 右击 省/自治区 -> 地理角色 -> 州/省/市/自治区 (左侧图标变成地球🌍)
  4. 双击 省/自治区
  5. 拖拽 省/自治区 到 颜色
  6. 拖拽 省/自治区 到 标签
  7. 拖拽 销售额 到 标签

树形图

  1. 点击左下角 新建工作表
  2. 重命名为 树形图
  3. 下拉：标记 -> 方形
  4. 拖拽 客户名称 到 详细信息
  5. 拖拽 销售额 到 大小
  6. 拖拽 订单日期 到 行
  7. 工具栏：标准 -> 整个视图
  8. 工具栏：突出显示 -> 客户名称
  9. 选择客户

智能显示

  1. 点击左下角 新建工作表
  2. 重命名为 智能显示
  3. 点击 子类别
  4. Ctrl(cmd) 点击 销售额
  5. 右上角：智能推荐 -> 气泡图
